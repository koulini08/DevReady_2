## DevReady: Smart Environment Health Checker

- A proactive tool built to keep your development environment clean, consistent, and ready to run.

## Techstack
- Frontend - HTML,CSS,TypeScript
- Backend - Python
- Database - SQLite

  ## The End Result

- DevGuard gives developers a complete safety net — scanning, detecting, predicting, and fixing — all before a single line of code runs.
- System Scanning
- Issue Detection
- Failure Prediction
- Suggested Fixes

## How to Run
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
